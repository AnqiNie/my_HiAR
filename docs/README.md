# HiAR Project Page

This is the GitHub Pages project page for HiAR.

## Setup

1. Enable GitHub Pages in repository settings, pointing to the `docs/` folder.
2. Add your figures to `static/images/`:
   - `method_overview.png` -- block-first vs step-first diagram
   - `pipeline_parallel.png` -- pipeline parallel scheduling diagram
3. Add sample videos to `static/videos/`:
   - `teaser.mp4` -- hero teaser video
   - `long_1.mp4` through `long_4.mp4` -- long video examples
   - `comparison_sf.mp4` -- Self-Forcing baseline
   - `comparison_hiar.mp4` -- HiAR result
4. Update author names, affiliations, and links in `index.html`.
